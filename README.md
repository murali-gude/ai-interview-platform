# AI Interview Platform

## Overview

AI Interview Platform is a full-stack web application that allows users to:

* Register and Login using JWT Authentication
* Create technical interviews
* Start interviews with generated questions
* Submit answers
* Receive automatic evaluation and scoring
* View interview history and previous results

## Tech Stack

### Backend

* Java 17
* Spring Boot
* Spring Security
* JWT Authentication
* Spring Data JPA
* MySQL
* Maven

### Frontend

* React.js
* React Router
* Axios
* CSS

## Features

### Authentication

* User Registration
* User Login
* JWT Token Security

### Interview Management

* Create Interview
* Start Interview
* Question Management

### Evaluation

* Automatic Answer Evaluation
* Score Calculation
* PASS / FAIL Status
* Feedback Generation

### Results

* Detailed Result Report
* Interview History
* Previous Attempts Tracking
